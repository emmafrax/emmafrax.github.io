Data Driven project
Community detection and visualization in signed networks

Individual project for the Data Driven Social Analytics class in the Master in Intelligent Interactive Systems (UPF).
This project aims to compare different community detection algorithms built for signed networks. The optimization of these algorithms can be helpful in the emergence of undetected groups in communities and in the study of polarization in social media. Signed networks allow capturing the positive interactions between nodes as well as negative, and create a challenge for both categorizing the nodes in the most coherent communities and visualizing such network in a clear layout. An extensive voting dataset from the European Parliament is used in order to exemplify these methods and analyse the results.
In this project we compare and apply to our dataset three different methods: the first one uses an approximation of the Potts model based on a continuous convex relaxation using hinge-loss functions, the second maximizes the modularity finding a community structure in the agreeing part of the network using the Louvain method and, finally, the third one uses spectral algorithms based on a discrete eigenvectors problem .
